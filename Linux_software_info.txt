Ядро Linux – центральная часть всей системы. Она выступает в качестве
посредника между физическим оборудованием компьютера и его программным
обеспечением, абстрагируя от пользователя и обычных программ сложную
низкоуровневую работу с физическим оборудованием компьютера, представляя
взамен простой, понятный и удобный в использовании интерфейс.

Util-linux – пакет утилит, в состав которых входит стандартный набор
служебных утилит командной строки, таких как утилиты для работы 
с файловой системой, консолью, разделами и сообщениями.

Glibc – пакет, содержащий стандартную библиотеку языка Си. Эта библиотека
предоставляет функции для выделения памяти, поиска каталогов, открытия и
закрытия файлов, чтения и записи файлов, обработку строк, соответствия
шаблонов (pattern matching), арифметические операции, и так далее.

Libstdc++ – Стандартная библиотека языка C++.

Binutils — набор инструментального ПО для обращения с объектным кодом в
объектных файлах различного формата.
 
Bash – командный интерпретатор, используемый в операционных системах
семейства Unix, в котором пользователь может либо давать команды
операционной системе по отдельности, либо запускать скрипты, состоящие из
списка команд.

Eudev – утилита, выполняющая функцию диспетчера устройств.

Sysinit — подсистема инициализации в Unix и ряде Unix-подобных систем,
которая запускает все остальные процессы.

GCC — набор компиляторов для различных языков программирования.

GDB — переносимый отладчик проекта GNU, который работает на многих UNIX-
подобных системах и умеет производить отладку многих языков
программирования, включая Си, C++, Free Pascal, FreeBASIC, Ada, Фортран и
Rust.

Make — утилита, автоматизирующая процесс преобразования файлов из одной
формы в другую.

Check — блиотека модульного тестирования для программ, написанных
на языке Си. 

Libelf — библиотека, используемая для доступа к объектным файлам. Она
позволяет читать, изменять или создавать файлы ELF в виде, не зависящем от
используемой архитектуры.

Python3 — высокоуровневый язык программирования общего назначения с
динамической строгой типизацией и автоматическим управлением памятью.

Autoconf — утилита для создания конфигурационных скриптов (configure),
которые автоматически настраивают пакеты с исходным кодом для работы в 
Unix-подобных операционных системах.

Automake — это утилита для автоматической генерации файлов Makefile.in из
файлов Makefile.am. Каждый файл Makefile.am фактически является набором
макросов для программы make (иногда с несколькими правилами). Полученные
таким образом файлы Makefile.in соответствуют стандартам GNU Makefile.
 
Vim — свободный текстовый редактор, созданный на основе более старого vi.
Ныне это мощный текстовый редактор с полной свободой настройки и
автоматизации, возможными благодаря расширениям и надстройкам.
 
Make — утилита, автоматизирующая процесс преобразования файлов из одной
формы в другую. Чаще всего это компиляция исходного кода в объектные файлы
и последующая компоновка в исполняемые файлы или библиотеки.
 
Iana-etc — пакет, содержащий данные для сетевых сервисов и протоколов.
Inetutils — набор сетевых утилит GNU.

Openssl — полноценная криптографическая библиотека с открытым исходным
кодом, широко известна из-за расширения SSL/TLS, используемого в веб-
ротоколе HTTPS.
 
IProute2 — это набор утилит для управления параметрами сетевых устройств 
в ядре Linux. 

Zlib – cвободная кроссплатформенная библиотека для сжатия данных.

Zstd – алгоритм сжатия данных без потерь.

Bc – интерактивный интерпретатор Си-подобного языка, позволяет выполнять
вычисления с произвольно заданной точностью.

Flex – генератор лексических анализаторов.

Shadow – содержит программ для управления пользователями системы и
групповыми базами данных.

Pkg-config – предоставляет унифицированный интерфейс для опроса
установленных библиотек с целью компилирования программного обеспечения
из его исходного кода. 

Ncurses – библиотека, написанная на языках Си и Ада, предназначенная для
управления вводом-выводом на терминал, в числе прочего, библиотека
позволяет задавать экранные координаты (в знакоместах) и цвет выводимых
символов.

Sed – потоковый текстовый редактор (а также язык программирования),
применяющий различные предопределённые текстовые преобразования к
последовательному потоку текстовых данных. 

Gettext – библиотека проекта GNU для интернационализации.

Grep – утилита командной строки, которая находит на вводе строки,
отвечающие заданному регулярному выражению, и выводит их, если вывод
не отменён специальным ключом.

Gdbm – база данных для хранения индексных файлов.

Gperf – это генератор идеальных хеш-функций из списка ключевых слов,
предоставляемых пользователем.

Gawk – C-подобный скриптовый язык построчного разбора и обработки входного потока (например, текстового файла) по заданным шаблонам (регулярным выражениям). 

Grub – загрузчик операционной системы GNU.

Bzip2 – бесплатная свободная утилита командной строки с открытым исходным кодом для сжатия данных.
